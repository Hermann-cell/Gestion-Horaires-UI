# 🎓 Gestion des Horaires — Dashboard Administratif

Application web moderne de gestion des horaires développée avec **React + Vite + Bootstrap**.

Interface professionnelle avec sidebar dynamique, navbar intelligente, contenu central évolutif et architecture modulaire prête pour la production.

---

## 🚀 Aperçu

- 📊 Tableau de bord avec statistiques
- 👥 Gestion des utilisateurs
- 🏫 Gestion des salles
- 👨‍🏫 Gestion des professeurs
- 📅 Planning hebdomadaire
- 🎨 Interface moderne et responsive
- 🔄 Routing dynamique avec React Router
- 🧩 Architecture modulaire scalable

---

## 🛠 Stack Technique

- ⚛️ React 18
- ⚡ Vite
- 🎨 Bootstrap 5
- 🧱 React-Bootstrap
- 🧭 React Router DOM
- 🎯 React Icons
- 💅 CSS personnalisé (variables + layout flexible)

## 📦 Installation

```bash
# Cloner le projet
git clone <repository-url>

# Aller dans le dossier
cd gestion-horaires

# Installer les dépendances
npm install
```

## 🧠 Architecture

Le projet utilise une architecture basée sur :

- 🔹 Layout global
- 🔹 Pages isolées
- 🔹 Composants réutilisables
- 🔹 Variables CSS centralisées
- 🔹 Routing imbriqué

Cette structure permet :

- Scalabilité
- Maintenance simplifiée
- Ajout facile de nouvelles fonctionnalités
